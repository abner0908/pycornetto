start_server(cdb_lu, cdb_syn, host='localhost', port=5204, log=None, verbose=False, max_depth=None, similarity=False, proxy_class=None)

source code 

main function to start the Cornetto XMLRPC server

Parameters:
  • cdb_lu (file or filename) - xml definition of the lexical units
  • cdb_syn (file or filename) - xml definition of the synsets
  • host (string) - host to listen on
  • port (int) - port to listen on
  • log (bool) - log requests
  • verbose (bool) - verbose output during parsing
  • max_depth (int) - maximal search depth (between 1 and 9)
  • similarity (bool) - extend interface with word similarity measures from SimCornet (requires cdb_lu file with counts)
  • proxy_class ((subclass of) CornetProxy) - class that serves as proxy to (a subclass of) the Cornet class (e.g. CornetProxy or SimCornetProxy)
